# SO-101 Calibration

## 개요

SO-101 로봇 팔의 **리더 암(Leader Arm)**과 **팔로워 암(Follower Arm)** 모두 Manual calibration을 진행해야 합니다. 캘리브레이션은 로봇 팔의 각 관절(joint)이 정확한 위치를 인식할 수 있도록 하는 중요한 과정입니다.

<Callout type="info">
캘리브레이션 이후 캘리브레이션 파일은 `~/.cache/huggingface/lerobot/calibration`에 생성됩니다.
</Callout>

## 캘리브레이션 과정

### 준비 사항

캘리브레이션을 시작하기 전에 다음 사항들을 확인해주세요:

- 리더 암과 팔로워 암이 모두 연결되어 있는지 확인
- USB 포트 연결 상태 확인 (`/dev/ttyACM0`, `/dev/ttyACM1`)
- 로봇 팔이 자유롭게 움직일 수 있는 공간 확보

### 캘리브레이션 순서

1. **리더 암 캘리브레이션** 먼저 진행
2. **팔로워 암 캘리브레이션** 이후 진행

---

## 1. 리더 암 캘리브레이션

### 캘리브레이션 명령어

다음 명령어를 실행하여 리더 암 캘리브레이션을 시작합니다:

```bash
python -m lerobot.calibrate \
    --teleop.type=so101_leader \
    --teleop.port=/dev/ttyACM0 \
    --teleop.id=leader
```

### 캘리브레이션 과정

1. **중간 위치 설정**
   명령어 실행 후 다음과 같은 메시지가 표시됩니다:
   ```
   Move None SO101Leader to the middle of its range of motion and press ENTER....
   Move all joints sequentially through their entire ranges of motion.
   Recording positions. Press ENTER to stop...
   ```

2. **관절 움직임**
   - 로봇 팔을 중간 위치(middle position)로 이동
   - 모든 관절(joints)을 순차적으로 전체 범위에서 움직여보기
   - ENTER 키를 눌러 기록 중지

3. **캘리브레이션 결과**
   캘리브레이션이 완료되면 다음과 같은 관절 위치 정보가 표시됩니다:

   ```
   -------------------------------------------
   -------------------------------------------
   NAME            |    MIN |    POS |    MAX
   shoulder_pan    |   1955 |   2007 |   3132
   shoulder_lift   |    709 |    724 |   2057
   elbow_flex      |   1605 |   3036 |   3110
   wrist_flex      |    808 |   2797 |   2803
   wrist_roll      |   2037 |   2048 |   2055
   gripper         |   2019 |   2046 |   3383
   ```

4. **파일 저장 확인**
   성공적으로 완료되면 다음 메시지가 표시됩니다:
   ```bash
   Calibration saved to ~/.cache/huggingface/lerobot/calibration/teleoperators/so101_leader/leader.json
   ```

---

## 2. 팔로워 암 캘리브레이션

### 캘리브레이션 명령어

리더 암 캘리브레이션 완료 후, 다음 명령어로 팔로워 암 캘리브레이션을 진행합니다:

```bash
python -m lerobot.calibrate \
    --robot.type=so101_follower \
    --robot.port=/dev/ttyACM1 \
    --robot.id=follower
```


### 캘리브레이션 과정

팔로워 암의 캘리브레이션 과정은 리더 암과 동일합니다:

1. 중간 위치로 로봇 팔 이동
2. 모든 관절을 전체 범위에서 움직이기
3. ENTER 키로 기록 완료
4. 관절 위치 정보 확인
5. 캘리브레이션 파일 저장 확인

---

## 캘리브레이션 파일

### 저장 위치

캘리브레이션이 완료되면 다음 위치에 파일이 저장됩니다:

- **리더 암**: `~/.cache/huggingface/lerobot/calibration/teleoperators/so101_leader/leader.json`
- **팔로워 암**: `~/.cache/huggingface/lerobot/calibration/robots/so101_follower/follower.json`

### 파일 구성

각 캘리브레이션 파일에는 다음 정보가 포함됩니다:
- 각 관절의 최소값(MIN)
- 현재 위치(POS)  
- 최대값(MAX)

<Callout type="success">
두 팔 모두 캘리브레이션이 완료되면 텔레오퍼레이션과 데이터 수집을 시작할 수 있습니다.
</Callout>

---

## 문제 해결

### 일반적인 문제

- **포트 연결 오류**: USB 포트 번호(`/dev/ttyACM0`, `/dev/ttyACM1`) 확인
- **권한 문제**: 사용자가 시리얼 포트 접근 권한을 가지고 있는지 확인
- **하드웨어 연결**: 로봇 팔의 전원과 USB 연결 상태 점검

### 캘리브레이션 재실행

캘리브레이션 결과가 만족스럽지 않다면 동일한 명령어로 다시 실행할 수 있습니다. 새로운 캘리브레이션 파일이 기존 파일을 덮어쓰게 됩니다.
